             Tic-Tac-Toe-Game
A Tic Tac Toe (XOX) game developed in Java using Swing for the graphical user interface.

✨ Features
•Two-player gameplay (X vs O)
•Neon-style dark UI design
•Score tracking
•Game reset after each round
•Scores are preserved between rounds


▶ How to Run
1-Clone the repository
2-Open the project in any Java IDE 
3-Run XoxInterface.java
